## CRITICAL: Orchestrator-Writes Pattern

**Subagents launched with `isolation: "worktree"` cannot use Write/Bash tools due to
permission hooks.** Do NOT instruct subagents to write files or commit. Instead:

- Instruct each agent to **return all file content as markdown code blocks** in their result
- The orchestrator (this session) reads the results and writes + commits each file directly
- This is the canonical pattern going forward
